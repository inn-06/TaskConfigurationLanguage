MyConfigConverter 
Конвертер конфигурационных файлов из пользовательского формата в TOML. Описание Утилита для преобразования конфигурационных файлов из специального формата (с поддержкой констант и таблиц) в стандартный формат TOML. Возможности:

Конвертация пользовательского формата в TOML
Поддержка констант с подстановкой значений
Обработка таблиц
Поддержка двоичных чисел (0b1010)
Обработка строк в одинарных кавычках
Фильтрация комментариев
Валидация входных и выходных путей
Логирование необработанных строк Требования
.NET 6.0 или выше
Операционная система: Windows/Linux/macOS Использование
Запустите программу: dotnet run --project ConfigurationLanguage1.csproj или dotnet run -- --input C:\Путь\Входных\Данных\Данные.txt --output C:\Путь\Выходных\Данных\Данные2.toml
Введите пути к файлам при запросе: Введите путь файла, который следует преобразовать C:\configs\input.conf Введите путь к файлу в который будет поступать выходная информация C:\configs\output.toml
Формат входного файла Поддерживаемый синтаксис: Константы: server_name := 'MyServer' max_connections := 100 binary_value := 0b1010 Таблицы: text table server([ host = 'localhost' port = 8080 ]) Присваивания log_level = 'debug' timeout = 30 Комментарии # Это однострочный комментарий {{!-- Это многострочный комментарий --}}

Основные классы и методы класс Program метод Main() - точка входа в программу метод Validation() - валидация входных параметров метод OutputRawStringsConsol() - вывод необработанных строк

Статический класс Parser Parse() - основной метод парсинга ParseValue() - парсинг значений ReplaceConstants() - подстановка констант FormatValue() - форматирование значений AppendToToml() - формирование TOML-структуры

Обработка ошибок Программа обрабатывает следующие ошибки:

Отсутствие входного или выходного файла
Неверное расширение выходного файла (должно быть .toml)
Ошибки чтения/записи файлов
Неподдерживаемые форматы данных
Логирование Программа сохраняет необработанные строки в StringBuilder CollectionRawStrings и выводит их в консоль при наличии.
